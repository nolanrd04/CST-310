# Checkered Triangles

Simple OpenGL/GLUT program that draws textured triangles and supports animation + keyboard controls.

## Build

```bash
g++ CheckeredTriangles.cpp -o CheckeredTriangles -lglut -lGLU -lGL
```

## Run

```bash
./CheckeredTriangles
```

## Keyboard Controls

- `p`: pause spinning
- `c`: continue spinning
- `u`: move up
- `d`: move down
- `l` or `L`: move left
- `r` or `R`: move right
- `+`: zoom in
- `-`: zoom out
- `Esc`: exit
