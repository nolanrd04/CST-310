# Specularcpp Demo

This project contains `specular.cpp`, a C++ OpenGL demo using FreeGLUT and GLSL Phong-style lighting.

## Build

```bash
g++ -std=c++17 specular.cpp -o specular -lglut -lGLU -lGL
```

## Run

```bash
./specular
```

## Controls

- `W A S D` move on the horizontal plane
- `Q E` move down and up
- `Arrow keys` look around
- `Esc` exit
